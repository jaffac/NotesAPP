# My Notes App 📝

https://notesapp-8by.pages.dev/

A sleek, dark-themed note-taking application built with SvelteKit and Supabase.

## 🌟 Features

- Google OAuth Authentication
- Create, Edit, and Delete Notes
- Dark Mode Styling
- Responsive Design

## 🛠 Technologies Used

- SvelteKit
- Supabase
- Tailwind CSS
- Google OAuth
